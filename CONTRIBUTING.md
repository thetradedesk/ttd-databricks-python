# Contributing to This Repository

Thank you for your interest in contributing to this repository. This repository is maintained exclusively by The Trade Desk and **we do not accept external pull requests or direct code changes**.

## How to Report Issues

If you encounter any bugs or have suggestions for improvements, please [open an issue](https://github.com/thetradedesk/ttd-databricks-python/issues) on GitHub using one of the provided templates:

- **[Bug Report](https://github.com/thetradedesk/ttd-databricks-python/issues/new?template=bug_report.yml)** — for unexpected behaviour or errors
- **[Feature Request](https://github.com/thetradedesk/ttd-databricks-python/issues/new?template=feature_request.yml)** — for new functionality or enhancements
- **[Question](https://github.com/thetradedesk/ttd-databricks-python/issues/new?template=question.yml)** — for general questions about usage

## Issue Triage

We will review and triage issues as quickly as possible. However, responses and resolutions are not guaranteed.

## Contact

If you have any questions or need further assistance, please feel free to reach out by opening an issue.

Thank you for your understanding and cooperation!

The Maintainers
